# Go API

A Go API built with Gin framework providing access.

## Getting Started

### Prerequisites

- Go 1.18 or higher

### Running Locally

```bash
# From project root
go run cmd/api/main.go
```

### API Endpoints

- `GET /health` - Health check endpoint

## Environment Variables

- `PORT` - Port on which the server listens (default: 8080)
```
